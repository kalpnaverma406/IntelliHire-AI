# IntelliHire AI

## Overview
IntelliHire AI is an AI-powered hiring assistant that helps candidates understand their skills through resume analysis and prepares them for interviews with AI-generated mock interview questions.

## Problem Statement
Many candidates struggle to identify their strengths and prepare effectively for interviews. Manual resume evaluation and interview preparation can be time-consuming and inconsistent.

## Solution
IntelliHire AI analyzes uploaded resumes, extracts key skills, and conducts AI-powered mock interviews to help candidates assess and improve their interview readiness.

## Features
- Resume Upload
- Resume Analysis
- Skill Extraction
- AI-Powered Mock Interview
- Interview Question Generation
- Candidate Feedback

## Tech Stack

### Frontend
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI Integration
- AI Model API

## How It Works

1. User uploads a resume.
2. AI analyzes the resume.
3. Skills are extracted and displayed.
4. AI generates interview questions.
5. Mock interview starts.
6. Feedback is provided to the user.

## Architecture

↓
